Stats
Live activity data — fully customisable
📊

Live Stats Display

Up to 6 stat circles arranged in a 3×2 grid, each showing 2 stats. Choose which stats appear in each circle from the settings. All values update live during recording.

  • Up to 6 circles, 2 stats per circle
  • Fully editable layout — choose what appears where
  • Available stats include distance, pace, time, heart rate, steps, elevation, ascent, descent and more

Import Tracks

  • Download GPX traces directly from OpenStreetMap
  • Import GPX files from your device storage
🗂️

Track List

Each track shows its name and file size. Tap any track to see its details and options.

  • Press the red ▶ play button to activate Nav — the route appears on the Record screen map
  • Save the track to your device as a GPX file
  • Upload to Strava (placeholder — coming soon)
  • Delete the track from the app

Page 4
Maps
Download and manage offline map files
🗺️

Download Maps

Browse and download offline OpenStreetMap regions. Once downloaded, maps work with no data connection.

  • Browse by continent and country
  • Some countries have a single map file, others are split into regions
  • Tap any region to download
📂

Import .map Files

  • Import any Mapsforge .map file from your device storage
  • Tap the red ✕ to delete a map and free up storage
Page 5
Route Planner
Plan and save routes directly in the app
✏️

Plan a Route

Tap the map to drop waypoints and build a route. The app calculates the path between points using OpenRouteService. Supports multiple activity profiles.

  • Tap the map to add waypoints along your route
  • Choose your profile — hiking, walking, cycling or car
  • View distance, estimated time and elevation profile as you plan
  • Tap the ↓ GPX button to save the route to your Tracks
  • Activate it for navigation straight from the Tracks page
