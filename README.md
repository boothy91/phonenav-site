# PhoneNav

Offline hiking and trail running navigation for Android phones.

## Status

**Currently in closed testing on the Play Store.**
DM on Reddit to join: [reddit.com/u/Due-Sand7582](https://www.reddit.com/u/Due-Sand7582)

## Features

- Offline OpenStreetMap maps — no internet required
- GPX route navigation with live position and remaining distance
- Record activities with stats: distance, pace, heart rate, steps, elevation
- Customisable stats screen — up to 6 circles, 2 stats each
- Import GPX routes or plan one in the built-in route planner
- Download maps by region for offline use, or import .map files
- Upload recordings to Strava *(placeholder — coming soon)*
- Light, Dark and Fjord themes

## App Pages

| # | Page | Description |
|---|------|-------------|
| 1 | Record | Start/stop/pause recording, map with location and compass |
| 2 | Stats | Live stats display, up to 6 customisable circles |
| 3 | Tracks | Import GPX, activate nav, save, delete, Strava upload |
| 4 | Maps | Download regions or import .map files |
| 5 | Route Planner | Plan a route on the map, save as GPX to tracks |
| 6 | Settings | All app preferences |

## Settings

1. Theme — Fjord, Light, Dark
2. Minimal Map — show only paths, waterways and roads
3. Distance — km or mi
4. Clock — 12hr or 24hr
5. Rec Distance — minimum distance before recording a GPS point (1m–20m)
6. Rec Time — minimum time between GPS points (1s–20s)
7. Import Colour — colour of imported tracks on the map
8. Record Colour — colour of the recording track on the map
9. Import Width — line width of imported tracks (1px–10px)
10. Record Width — line width of the recording track (1px–10px)
11. Lap Splits — vibrate and show split time every set distance *(untested)*
12. Lock Zoom — fixed zoom level when location is locked
13. Keep Active — prevent screen sleep during recording
14. Nav Distance — show remaining route distance in notification when nav is active
15. Nav Elevation — show remaining elevation when nav is active *(not yet shown)*
16. Heart Rate — enable HR recording
17. Steps — enable step recording
18. Countdown — countdown before recording starts (off, 3s, 5s)
19. Exit App
