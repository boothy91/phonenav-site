# PhoneNav

Offline hiking and trail running navigation for Android phones.

## Features

- Offline OpenStreetMap maps — no internet required
- GPX route navigation with turn-by-turn tracking
- Record activities with stats: distance, pace, heart rate, steps, elevation
- Import GPX routes or download from the built-in planner
- Download maps by region for offline use
- Light, Dark and Fjord themes

## Download

[Download latest APK](https://github.com/boothy91/phonenav-site/raw/main/PhoneNav.apk)

## Installation

1. Download the APK above
2. On your Android phone, enable "Install from unknown sources" in Settings
3. Open the downloaded APK and install

## Companion App

Pairs with [WristNav](https://github.com/boothy91/wrist-nav-site) for Wear OS watches.

## Requirements

- Android 8.0 or higher
